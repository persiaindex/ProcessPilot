# ProcessPilot - Day 1 Notes

## Job description to feature mapping
- Process digitization and automation -> request form, workflow, audit trail
- React apps communicate with Django via REST -> React frontend + DRF backend
- Container-based architecture -> Dockerized frontend, backend, and database
- AI/ML modules -> optional categorization service
- Automated simulations -> optional mock external processing flow

## User roles
- Employee: creates and tracks requests
- Reviewer: approves or rejects requests
- Admin: oversees departments and access
